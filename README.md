# Projeto SQL – Mini-mundo 

Este repositório contém scripts SQL completos para criação e manipulação de dados em um banco relacional baseado no minimundo de cursos online.

## Estrutura
- 01_create_tables.sql — Criação das tabelas
- 02_insert_data.sql — Inserts iniciais
- 03_select_queries.sql — Consultas SELECT
- 04_update_examples.sql — Exemplos de UPDATE
- 05_delete_examples.sql — Exemplos de DELETE

## Como usar
1. Importe o arquivo 01_create_tables.sql no MySQL Workbench/PGAdmin.
2. Execute o arquivo 02_insert_data.sql.
3. Use os demais scripts para consultas, atualizações e exclusões.
